●EnjoiCryptoの概要

EnjoiCrypto は Windows/Linux のどちらでも動作するよう開発された、パスワードを用いたファイル暗号化プログラムです。
暗号化されたファイルは exe 形式に変換され、 .NET/Mono 環境で復元が可能です。
暗号化に使用されているアルゴリズムは多分 AES です。
Windows/Linux のどちらでも復号化できる exe 形式のものが欲しかったので作りました。

・パスワードを用いたファイルの暗号化
・exe 形式のファイルを出力
・Windows/Linux どちらでも復号化が可能(.NET/Mono のどちらかが必要)
・コマンドラインによる対話形式


●インストール方法

アーカイブ内容を展開して、適当なディレクトリを作ってそこに内容を移動しておいてください。
クリックをしても起動して使えますが、コマンドライン形式です。


●アンインストール方法

インストールしたファイルをすべて消してください。
レジストリ等は一切使用しておりません。
